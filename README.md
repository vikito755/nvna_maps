# nvna_maps

Description: Sample of how Google maps can be impelemnted naval-acad website.

Code:

index.html - HTML file to display the anchor tags, the "div" element with id "container" is connected to the javascript.

scripts.js - 
1. Waits for the HTML to load, then loads the "locations" array which will be substituted for a ".json" file.
2. For each location in the array creates an anchor tag with the appropriate coordinates.
- The variables are just for readability and maintainability of the code.
- There is room for optimisation.

styles.css - makes the tags readable - likely won't be needed for integration on the websiteon the website.
